# Collaborative Projects
AVAnnotate provides different options for collaboration, which include:

### AVAnnotate Collaborators
Users can add collaborators directly in AVAnnotate. Visit [Adding AVAnnotate Collaborators](https://avannotate.github.io/documentation/pages/add-collab/){:target="_blank" rel="noopener"} for more information and detailed instructions on adding collaborators in the AVAnnotate application.

### GitHub Collaborators
Users can also add and organize collaborators for AVAnnotate projects in GitHub. Visit [Managing GitHub Collaborators](https://avannotate.github.io/documentation/pages/gh-collab/){:target="_blank" rel="noopener"} for detailed instructions on creating a GitHub Organization and adding collaborators through GitHub. 

AVAnnotate projects with collaborators will appear on the `Shared Projects` tab of the `Projects` page.
