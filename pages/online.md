## Creating Online Projects
Select `+ Add` and input information about the new project, including: 
- `Title`
- `Description`
- `Language`
- `Project Slug` that will appear in the URL.
- `Project Authors`
- `Collaborators` GitHub collaborators can be searched via AVAnnotate.
